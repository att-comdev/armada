Manifest Exceptions
===================

| Exception                        | Error Description                         |
+==================================+===========================================+
| ManifestException                | An exception occurred while attempting to |
|                                  | build an Armada manifest. The exception   |
|                                  | will return with details as to why.       |
|                                  |                                           |
|                                  | **Message:**                              |
|                                  | *An error occured while generating the    |
|                                  | manifest: <details>*                      |
|                                  |                                           |
|                                  | **Troubleshoot:**                         |
|                                  | *Coming Soon*                             |
